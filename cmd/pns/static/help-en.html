Formatting

Notes in PNS are written in so-called Markdown format or actually in its dialect called CommonMark. This is a lightweight markup, that is it looks almost like plain text but some characters have special meaning triggering some kind of formatting.

There are two kinds of formating: inline formating which can be used inside any paragraph and paragraph formating which determines division of the text into headings, paragraphs, ordered and unordered lists, tables, and preformatted blocks of text.

Keyboard shortcuts

PNS pages can be navigated using keyboard shortcuts.

View mode

On the pages that display notes the following shortcuts are available

Key Action
Alt + n select next note (just n will work if some note is already selected)
Alt + p select previous note (just p will work if some note is already selected)
Alt + l switch between search field and current note (just l will work if not in search field)
Alt + a add new note (just a will work unless search field is selected; works as “Add note” button)
e edit selected note (works as “Edit” link)

Editing mode

On the note editing/adding pages the following shortcuts are available

Key Action
Alt + l switch between tag editing field and note editing field
Alt + r reload preview (works as “Preview” button)
Alt + s submit note (works as “Submit” button)

Common shortcuts

These shortcuts work in Firefox and Chrome

Key Action
Alt + ← back to previous page
Alt + → forward to next page

Topics and tags

Each note must be connected with one or more topics and tags (names of topics start with a slash, such as /mytopic, and names of tags start with a letter, such as mytag). PNS starts with and index page containing list all of the defined topics and tags. You can click on a particular topic or a tag to view notes connected with this topic or tag.

On the left side of the top panel (in the view mode) there are buttons for the currently selected topic and tags (only one topic may be selected at a given time but more than one tag). Only notes that are connected with selected topic (if any) and selected tags (if any) are displayed. Clicking on any of these buttons will remove the given topic or tag from the selection.

Each note has a footer listing all topics and tags it is connected to. Clicking on any of its topics and tags in the footer will add this topic or tag to the selected ones (if a topic is clicked and another topic is already selected the new topic will replace the old one in the selection).

Selected topics and tags may also be changed using the search field. Search field may also be used for full text search in the notes. Here are example searches:
